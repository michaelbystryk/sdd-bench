## Rubric — five dimensions, each scored 0–5 (half-points only for true between-anchor cases)

Score against ABSOLUTE anchors. Do NOT grade on a curve or inflate. A competent,
unremarkable professional answer is a 3, not a 5. 5 is reserved for genuinely excellent.

### 1. Correctness — is what it says true and technically sound
0 central claims wrong (would misdirect a team) · 1 multiple significant errors · 2 mostly sound but ≥1 significant error a domain reviewer would flag · 3 sound throughout, only minor imprecisions · 4 sound + precise, claims appropriately qualified, no errors on careful review · 5 above + correctly handles the subtle/contested points.

### 2. Coverage — did it address the whole problem
0 a sliver · 1 major required areas missing · 2 ~half covered or all only nominally · 3 all required areas substantively covered · 4 above + engages implicit/edge concerns · 5 above + surfaces material considerations the brief didn't hint at that a senior reviewer would agree belong.

### 3. Insight depth — non-obvious findings
0 generic boilerplate · 1 restates the brief's facts · 2 competent but entirely predictable · 3 ≥1 genuinely non-obvious finding/connection/trade-off specific to this problem · 4 several non-obvious findings, second-order effects engaged · 5 above + a finding that reframes the problem or would change the buyer's decision.

### 4. Actionability — could a team execute from this
0 none · 1 vague · 2 concrete but unprioritized/missing sequencing · 3 concrete, prioritized, scoped — start tomorrow · 4 above + costs/risks/effort honestly stated · 5 above + explicit decision points, success measures, what-would-change-my-mind.

### 5. Communication — is it a usable professional document
0 unusable · 1 hard to navigate, wrong altitude · 2 readable but bloated/repetitive OR structured by process rather than reader need · 3 clear structure, appropriate length (per the brief's stated band), conclusions findable in one pass · 4 above + tight, every section earns its place · 5 above + a director would forward it unedited.

**Length-band rule:** the brief states a target length. Exceeding it is scored DOWN in
Communication (bloat), never rewarded as Coverage — a longer document does not buy coverage.

**Planted-truth note:** this deliverable is scored not only on the rubric but on RECALL
(did it find the keyed items in the answer key, per each item's minimum-credit bar — a
floor; misses are the signal) and PRECISION (did it confidently assert findings that match a
named decoy the key rules out, or hallucinate findings with no basis — false positives lose).
Score recall and precision per the answer key included in this file. Recall is NOT folded
into the /25 rubric sum; report it separately.
